Le mur

      

        « There is no API, socket, or pipe to inject a prompt into an active session. »
      

      

        
01 TIOCSTI est mort — désactivé par défaut sous Linux 6.2, root-only sur macOS, supprimé d'OpenBSD.

        
02 Écrire dans le PTY slave part vers l'affichage, jamais dans le stdin du process.

        
03 Seul un programme qui détient le PTY maître peut injecter. C'est là que csend opère.

En pratique

      
Deux verbes. Tout le reste suit.

      

        

          
Dépose & relève (coopératif — tout OS)

          
# une session écrit à une autre, sans cmux, partout
csend inbox SACEM "lance le build de prod"
# la cible relève son inbox
csend recv

        

        

          
Pilote la flotte (state-aware)

          
csend list        # sessions + état (idle/busy/confirm)
csend tree        # graphe familial père → enfants
csend send --down "go"  # broadcast aux enfants

        

        

          
Identité & recovery par seuil

          
csend id --create        # vault chiffré (passkey à venir)
csend recovery split 2 3  # 3 parts, seuil 2-sur-3
# perds un device sans perdre le vault

        

      

    


    
    

      

        
Sécurité

        
Chiffré par conception.
Primitives auditées, jamais maison.

      

      
    
        
  • Hybride PQC
    X25519 ⊕ ML-KEM-768 : il faut casser les deux KEM pour lire. Anti « Harvest Now, Decrypt Later ».
    • 
        
  • Vault à passkey
    Clés au repos en AES-256-GCM, déverrouillage WebAuthn/FIDO2 — MFA résistant au phishing.
    • 
        
  • Recovery Shamir
    Secret découpé N-sur-M sur GF(2⁸). K-1 parts ne révèlent rien — sûreté de seuil.
    • 
        
  • Tout en stdlib Go
    Ed25519, ML-KEM, AES-GCM, HKDF, PBKDF2 — zéro dépendance externe à auditer.
    • 
      

    


    
    

      
Installer

      
Pour les humains. Et pour les agents.

      

        

          
Homebrew macOS · Linux

          
brew install aissablk1/tap/csend

        

        

          
Go toute plateforme

          
go install github.com/aissablk1/csend@latest

        

        

          
Script audite-le d'abord

          
curl -fsSL https://csend.dev/install.sh | sh

        

        

          
Source zéro confiance

          
git clone github.com/aissablk1/csend
cd csend && make install
